# Features

- Add todo
- Mark todo as done
- delete todo
- realtime connection

# SetUp

```
in /utils/db.js 
Replace your-mongodb-uri and with your actual MongoDB URI.
```

# Installation
## Install backend dependencies
```
cd backend
npm install
```

# Install frontend dependencies
in another tab
```
cd frontend
npm install
```
# Run Application
## Run frontend (:3000)
```
cd frontend
npm start
```
##  Run backend (:5173)
```
cd backend
npm index.js
```
